网站迁徙日志

Nov 25 2018

My website has been deployed on Google Cloud Platform for almost a year, because the network blockade, its access speed is extremely slow by ipv4. These days, I learned about Github Pages and I decide to move my website to Github Pages. After doing some necessary searches, I chose Hexo. It seems that Github Pages recommend Jekyll, however, Windows is not an officially-supported platform.

Install Hexo is easy, what I need to do is to take some time to convert previous HTML files to Markdown files.
